# Invoice Generator Project Plan

Planning documents untuk Simple Invoice Generator webapp.

## Files

| File | Purpose |
|------|---------|
| `PLAN.md` | Master plan lengkap (tech stack, schema, components, API) |
| `CHECKLIST.md` | Implementation checklist dengan tracking progress |
| `QUICKSTART.md` | Quick reference untuk memulai session baru |

## Project Summary

**Goal:** Simple invoice generator untuk freelancers, $0 cost

**Stack:**
- Next.js 14 (App Router, TypeScript)
- Supabase (Database, Auth, Storage)
- @react-pdf/renderer (PDF generation)
- Tailwind CSS (Styling)
- Resend (Email)
- Vercel (Hosting)

**Timeline:** ~3-4 weeks untuk MVP
